### Лабораторная работа 1: Основы UDP. Эхо-сервер и клиент

Цель: Освоить основы работы с UDP-сокетами, создать простейший сервер, который возвращает полученное сообщение обратно клиенту.

Задание:
1. Написать UDP-сервер, который:
    - Принимает датаграммы от клиента.
    - Выводит полученное сообщение и адрес клиента в консоль.
    - Отправляет это же сообщение ("эхо") обратно клиенту.
2. Написать UDP-клиент, который:
    - Считывает строку с клавиатуры.
    - Отправляет её серверу.
    - Ожидает и выводит ответ от сервера.

Теоретическая часть: Сокеты, socket(), bind(), sendto(), recvfrom(), sockaddr_in.

---

### Описание реализации
Реализация состоит из двух исполняемых файлов: server и client. При запуске сервера передается порт сервера, который он будет слушать:
```
./server [порт]
```
Клиент при запуске принимает два аргумента: адрес сервера и порт, который тот слушает:
```
./client [адрес сервера] [порт]
```
Клиент так же принимает udp-пакеты, открывая сокет на порту 22334 (см **defs.hpp**). Далее, пользователь может вписывать с клавиатуры строки длиной не больше 1023 символа. Строка посылается на сервер, тот отвечает той же строкой, посылая дейтаграмму на порт 22334. Если ввести строку `exit`, обе программы завершат свою работу.

### Сборка 
В папке `udp`:
```
make
```
